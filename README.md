# Healthcare-Diabetes-Data-Analytics-Pipeline
End-to-End Data Analytics project automating healthcare data pipeline using Python, SQL Server, and Power BI.

## 🛠️ Tech Stack & Analytical Ecosystem
* **ETL Ingestion Engine:** Python (Pandas)
* **Relational Database Server:** Microsoft SQL Server (SSMS)
* **Business Intelligence Tool:** Power BI Desktop

---

## 📊 Core Architecture Workflow
1. **Python Data Engine:** Generates and processes custom patient medical variables tracking parameters matching primary constraints schema layout metrics.
2. **SQL Relational Layer:** Tables split layout links mapping setup logic calculations metrics using continuous relational joins and Dynamic Server Views (`View_KPI_To_Win_Master`).
3. **Power BI Visual Representation:** Configured dynamic Line & Stacked Combo Charts tracking patient age distribution ranges and `PredictiveRiskIndex` target thresholds matrix tracking.

---

## 💡 Analytics Value Delivered
* Built an automated loop tracking database records scalability metrics instantly from 35 rows up to 65 records pipeline check logs.
* Resolved .NET Provider factories conflict data mappings optimization issues logic structure.
* Standardized dashboard age grouping buckets structure layout metrics.
